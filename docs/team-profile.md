# 关于团队

## 团队故事

我们小组一共有八位成员，是根据共同的理想以及兴趣爱好聚集了在一起开发实现这个有余（youyu）众包系统。在这个项目我们一共分配了三个大块去完成这个系统——产品经理、前端开发、后端开发。

而整个进程中最先开展主要工作的是产品组和UI设计（这一部分，主要交给前端来完成）之后前端和后台也陆续开展了工作，接着大家在通过开会商讨，并且不断发现问题、克服困难，最终完成了这个"有余"系统的开发。
